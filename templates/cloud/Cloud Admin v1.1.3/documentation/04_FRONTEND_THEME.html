Welcome to Cloud Admin Frontend

This section describes how to tweak, use & modify the frontend theme.

Files and folder structure

The files are arranged in the following generic structure. Please note that details regarding the contents of each folder is discussed under respective topics.

  1. bootstrap - contains the bootstrap source files.
  2. css - contains all the CSS files.
  3. font-awesome - contains the Font-Awesome 4.0.1 files
  4. img - contains all the images
  5. js - contains all the javascript files
  6. less - contains all the required LESS files. (For future proofing your template)
  7. index.html - the index page/homepage.

Structure of HTML files

The basic structure of the HTML theme is as follows

<!DOCTYPE html>
<html lang="en">
<head>
<meta http-equiv="content-type" content="text/html; charset=UTF-8">
<meta charset="utf-8">
<title>Simplifly Responsive Bootstrap Theme - Home</title>
<meta name="viewport" content="width=device-width, initial-scale=1.0">
<meta name="description" content="">
<meta name="author" content="">
<!-- STYLESHEETS -->
<!-- FONTS -->
</head>
<body>
    <!-- PAGE -->
    <div id="page">
    </div>
<!-- JAVASCRIPTS -->
</body>
</html>

Everything goes inside the <div> named page.

Including fonts

Include fonts under <head> as follows

<link href='http://fonts.googleapis.com/css?family=Lato:200,400,700' rel='stylesheet' type='text/css'>

Changing the logo

To change the logo

  1. Create a new logo preferably in 4x1 ratio and in transparent PNG for best display E.g. 400x100
  2. Go to the folder Simplifly - img - logo
  3. Save your logo as logo.png

Changing the header background cover

To change the background cover

  1. Go to the folder Simplifly - img
  2. Save your new image as home-bg.jpg

Adding/Editing the top navigation bar

The top navigation should be edited in three places viz.

Mobile Menu

  • One that starts with <div id="mobile-menu" class="list-group collapse text-center">
  • Add a new item by adding <a href="#portfolio" class="list-group-item">Portfolio</a>

Regular Top Menu

  • One that starts with <!-- HERO --><div class="container-transparent text-center">
  • Add a new item by adding <li><a href="#parallax-1">Demo</a></li> either to heronavleft or right.

Scroll down visible always on top menu

  • One that starts with <nav id="fixed-top-navigation">
  • Add a new item by adding <li><a href="#portfolio">Portfolio</a></li>

Playing with font sizes

The theme provides an easy way to emulate the uppercase large font size that is on the homepage.

  • class="bigintro" for hero text like effect
  • class="big for products headline like effect
  • class=text-center to center align

Playing with colors

The theme provides the following 4 generic color swatch which can be applied on any valid HTML element.

  • Primary class="color-primary"
  • Secondary class="color-secondary"
  • Light class="color-light"
  • Dark class="color-dark"

Adding patterns

Two type of patterns can be added viz. lines and dots

  • Lines class="pattern"
  • Dots class="dots"

Changing Font-Awesome icons

Font Awesome 4.0.1 is a great way of adding glymphs that scale no matter what. They are beautiful. To add a new icon

<i class="fa fa-check"></i>

To scale add the following

<i class="fa fa-check icon-2x"></i>

2x, 3x, 4x, 5x etc. The list of all icons can be found here: Full Icons List

Playing with white spaces

We have provided a quick and easy way to add spaces in the theme. Just add the following

<div class="divide60"></div>

to add 60px space.

Similarly, the valid classes are divide40, divide55, divide60, divide85 and divide90

Our Products Section

The product section is powered by jQuery Isotope. Mention the categories under the section <div id="filter-controls" class="btn-group">

<div id="filter-controls" class="btn-group">
  <div class="hidden-xs">
      <a href="#" class="btn btn-default" data-filter="*">All</a>
      <a href="#" class="btn btn-info" data-filter=".category_1">Android Apps</a>
      <a href="#" class="btn btn-danger" data-filter=".category_2">iPhone Apps</a>
      <a href="#" class="btn btn-success" data-filter=".category_3">Windows Apps</a>
      <a href="#" class="btn btn-warning" data-filter=".category_4">Web Apps</a>
  </div>
  <div class="visible-xs">
       <select id="e1" class="form-control">
            <option value="*">All</option>
            <option value=".category_1">Android Apps</option>
            <option value=".category_2">iPhone Apps</option>
            <option value=".category_3">Windows Apps</option>
            <option value=".category_4">Web Apps</option>
        </select>
  </div>
</div>

Then mention the category item contents under <div id="filter-items" class="row">

A single category item content is as follows:

<div class="col xs-12 col-sm-4 category_1 item">
<div class="filter-content">
    <img src="img/gallery/1.png" alt="" class="img-responsive" />
    <div class="image-content">
        <h4>Image Title</h4>
        <p class="hidden-xs hidden-sm">
            For faster mobile-friendly development, use these utility.
        </p>
        <a href="#" class="btn btn-sm btn-warning">Know More</a>
    </div>
</div>

- Here you can change the image src.

Editing Testimonials section

The testimonial section is a carousel that can contain any number of items. The basic markup is as follows

<div class="carousel-inner">
<div class="item active">
    <div class="container">
        <div class="carousel-caption">
        <div class="row">
            <div class="circular col-md-6">
                <img src="img/testimonials/headshot1.jpg" alt="headshot #1" />
                <h3>John Doe</h3>
                <p>Testimonial text goes here."</p>
                <h4>www.techfirst.com</h4>
            </div>
            <div class="circular col-md-6">
                <img src="img/testimonials/headshot2.jpg" alt="headshot #2" />
                <h3>Jane Doe</h3>
                <p>Testimonial text goes here."</p>
                <h4>www.techsecond.com</h4>
            </div>
        </div>
        </div>
    </div>
</div>
</div>

Simply replace the images, testimonial text and company name. Any valid HTML can go inside. You can put your images in Cloud Admin - frontend_theme - img - testimonials just to maintain uniformity.
